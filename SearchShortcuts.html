 Accesos directos para el menú de búsqueda

 Accesos directos predeterminados

 A continuación se muestra una lista de atajos de teclado predefinidos y accesos directos del mouse para Notepad ++ . Se pueden configurar libremente utilizando Atajos de teclado ... , así como también cualquier comando que no esté vinculado directamente a atajos predeterminados. Esta lista no incluye los accesos directos estándar proporcionados por los complementos de Notepad ++ .

 menú Buscar
 atajo  acción
 Ctrl-F  Lanzar el diálogo de búsqueda
 Ctrl-H  Lanzar el cuadro de diálogo Reemplazar
 F3  Busque el próximo
 Shift-F3  Busque el anterior
 Ctrl-Shift-F  Buscar en archivos
 F7  Saltar a la ventana de resultados de búsqueda
 Ctrl-Alt-F3  Siguiente búsqueda volátil
 Ctrl-Alt-Shift-F3  Búsqueda volátil anterior
 Ctrl-F3  Seleccione y encuentre lo siguiente
 Ctrl-Shift-F3  Selecciona y encuentra el anterior
 F4  Siguiente resultado
 Shift-F4  Resultado anterior
 Ctrl-Shift-I  Búsqueda incremental
 Ctrl- n  Siguiente aparición de estilo N ( N es 1 5 o 0 para el estilo predeterminado)
 Ctrl-Shift- N  Incidencia anterior del estilo N ( N es 1 5 o 0 para el estilo predeterminado)
 Ctrl-F2  Alternar marcador
 F2  Ir al siguiente marcador
 Shift-F2  Ir al marcador anterior
 Ctrl-B  Ir al delimitador simétrico (disponible si el cursor está en un delimitador)
 Ctrl-Alt-B  Seleccione todo entre Delimitadores simétricos (disponible si el cursor está en un delimitador)
