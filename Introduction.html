introducción

Dedicamos este documento a las innumerables horas perdidas no solo por aquellos que buscaron la respuesta a preguntas simples, sino también por las personas que respondieron preguntas que no deberíamos tener. preguntar La documentación es un arte difícil, y una buena documentación es prácticamente inaccesible =].

Todas las imágenes utilizadas en este archivo de ayuda se toman con la configuración predeterminada. Si modifica uno de estos parámetros, tendrá un desplazamiento con la imagen, que es especialmente visible con la barra de herramientas (la posición de las imágenes siempre será la misma).

Los botones del mouse se utilizan para una persona diestra. Sin embargo, para la configuración para zurdos, deberá invertir el uso de los botones. Las acciones ordinarias en Notepad ++ se basan esencialmente en el uso estándar de Windows.

De la misma manera, la visualización estándar se justifica de izquierda a derecha. Con algunas ubicaciones (como H breu o Farsi), Notepad ++ puede usar justificación inversa.

Este archivo de ayuda usa etiquetas para distinguir cierta interfaz o elementos de interacción.

Cuando se puede seleccionar una opción en el menú (abra el menú y luego seleccione la opción), se verá así: Menú-> Opción
donde la flecha indica 'Opción' en un submenú "Menú"

A veces, las etiquetas se utilizan en la GUI para describir un elemento cuando no pueden tener ningún texto asociado directamente. Las etiquetas se ven así: etiqueta
Finalmente, cuando se requiere el mouse o la interacción con el teclado, los botones o teclas que se presionarán se marcan como: Leftmousebutton

Este marcado se basa en la apariencia predeterminada de Microsoft Windows. Esto puede ser diferente en su propio sistema, pero los nombres (texto) son siempre los mismos. Esto es especialmente importante en las capturas de pantalla. Son solo el título de referencia, su apariencia de Notepad ++ puede ser muy diferente.
