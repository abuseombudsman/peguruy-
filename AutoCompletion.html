Entrada semiautomática

Notepad ++ tiene capacidades de autocompletar, lo que le permite terminar una palabra a medida que escribe dando una lista de opciones posibles o insertarla de inmediato. si solo hay uno disponible (proporcionado por el componente Scintilla ).

    Hay dos tipos de autocompletado.
  • El primero usa una lista de palabras clave predefinidas y se conoce como la finalización de las funciones . Se activa seleccionando  Editar -> Autocompletar -> Comp. función o automáticamente cuando se escribe una cierta cantidad de caracteres (ver también las Preferencias ).
  • El segundo, Completar palabra , recupera todas las palabras en el archivo actual y ayuda a completar la palabra. Esto se habilita seleccionando  Editar -> Autocompletar -> Completar palabra , o ingresando un número de serie de la lista (ver Preferencias ). Word Completion es el único tipo que puede terminar automáticamente una palabra sin mostrar primero una lista de opciones.
  • Ambas fuentes de finalización semiautomática están disponibles por separado o juntas.

Un tema relacionado (bastante similar) es la visualización de los parámetros de la función o ayuda con las llamadas a funciones . Esta es una pequeña información sobre herramientas que muestra los parámetros necesarios (y tal vez algunos detalles sobre ellos) para una función en la que se está escribiendo una llamada. Estas son solo indicaciones para la programación, que no se utilizan para ninguna otra tarea. La información de burbuja resaltará el parámetro que se está editando y cualquier información útil sobre el parámetro. Es puramente informativo y no hará ninguna corrección del texto. Para activar la información sobre herramientas (si existen datos), seleccione  Editar -> Autocompletar -> Información masiva de los parámetros de la función .

Completar está vinculado a un idioma finalizado, a excepción de la finalización de palabras (ver idiomas ) y los llamados archivos API para cada idioma. Para obtener más detalles sobre estos archivos XML y cómo hacer los suyos, consulte NpWiki ++ .

También es posible completar el directorio absoluto o las rutas de archivos desde Notepad ++ (acceso directo de forma predeterminada: Ctrl-Alt-Space ).
