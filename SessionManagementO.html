La gestion de session

 Les sessions sont un ensemble de fichiers ouverts dans Notepad + +. Ils stockent les fichiers ouverts, le fichier actif (et la vue, consultez la section Vue-Multiple), la sélection actuelle et la position dans le dossier, les signets en cours (voir Signets ) et le langage courant (voir la section sur les langages). Utiliser une session, c'est ouvrir un ensemble (relatif) de fichiers en une seule action.
