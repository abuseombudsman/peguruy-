 Extension du menu contextuel sous Windows

 Par défaut, le programme d'installation de Notepad++ est livré avec une extension de "Shell". Elle ne fait pas vraiment partie du programme, mais est très utile. Si vous choisissez de l'installer, puis faites un clic droit sur n'importe quel fichier, vous obtiendrez une option supplémentaire nommée "Edit with Notepad++" avec l'icône de Notepad++.

 Notez bien que pour installer ou supprimer cette extension, il est parfois nécessaire de relancer l'Explorer. Pour cela, il vous sera nécessaire de redémarrer Windows, ou de quiter votre session et d'en rouvrir une. On peut aussi utiliser le gestionnaire de tâches pour terminer explorer.exe. Vous pouvez le faire en appuyant sur Ctrl + Alt + Shift + Del.

 Si l'extension de Shell est déjà installée, l'installateur de Notepad++ ne pourra pas le mettre à jour ou l'écraser, vous devez d'abord le désinstaller et redémarrer Explorer. Si vous émettez la commande d'enregistrement de DLL (regsvr32 /i NppShell_05.dll), une boîte de dialogue apparaîtra à partir de laquelle divers détails du comportement de l'extension peuvent être modifié. Le nombre de possibilité augmente à mesure que les versions croissent. Remplacez "01" par le numéro le plus élevé de fichier que vous trouverez. A noter que, sur de plus vieilles versions de Notepad++, il faut remplacer "_05" par "_04" ou moins.
