 Plegado / desplegado de texto

 Notepad ++ admite doblar / desplegar para sus documentos de texto. Esto significa que algunas líneas de su texto pueden estar ocultas como marcas en el margen. Funciona bien para lenguajes estructurados como C ++ o XML. Los bloques de texto están jerárquicamente vinculados entre sí y una vez plegados (lo decimos también), solo la primera línea permanece visible, si no están dentro un bloque padre doblado. Este dispositivo es eficaz para echar un vistazo rápido a su documento y mostrar solo la parte más interesante del documento. Al desenrollar el texto, aparece de nuevo en su totalidad.

 Puede reconocer el texto que se puede doblar mirando el margen de la izquierda. Este margen muestra iconos y líneas para indicar la estructura del texto (se puede cambiar la apariencia en el margen, consulte Mejorador de sintaxis y Preferencias ). El analizador léxico actual determina la estructura (ver idiomas ). Por defecto, los bloques están representados por cuadrados y líneas para indicar la estructura de los pliegues. Un nodo con un signo menos ( ) indica que el texto puede colapsarse. Por el contrario, un nodo con un signo más () indica que el texto puede expandirse. Un texto colapsado se indica con una línea horizontal encima de la primera línea del texto plegado , que siempre estará visible. Las líneas marginales limitan las líneas de texto colapsables. Las líneas verticales (  ) que indican los grupos de líneas retráctiles juntas terminan en una pequeña línea horizontal ( ) o cuadro.

    Hay varias formas de doblar texto.
  •  La forma principal es hacer clic con el botón izquierdo del mouse en un cuadro con un signo menos. Esto dobla todo el texto en este grupo.
  • Lo mismo con presionar la tecla Ctrl para doblar el grupo, pero también todos los subgrupos dentro. Esto significa que cuando abra este grupo, todos los subgrupos estarán más abajo.
  • Usando el menú:   Ver -> Plegar todo se  plegará cada grupo en el documento, minimizando la mayor cantidad de texto posible.  Ver -> Plegar el nivel actual  solo retrae el grupo actual (el grupo que contiene la línea actual, en cualquier parte del grupo). Esto no replicará los subgrupos.  Pantalla -> Nivel de plegado ... omitirá el nivel seleccionado (de 1 a 8). Esto replica grupos a un nivel particular, pero no a todos los subgrupos. Puede usar esta opción para mostrar su texto con un "nivel de detalle".
    Para desplegar el texto del doblez , realice una de las siguientes acciones:
  • Haga clic en el cuadro con el signo más en el margen de plegado, mostrando el contenido de ese grupo.
  • Haga clic en el cuadro con el signo más con la tecla Shift o Ctrl . Esto desplegará los subgrupos de grupos y pliegues.
  • Usando el menú:   Ver -> Desconectar Todo se  expandirá cada (sub) grupo en el documento, haciendo visible la cantidad máxima de texto.  Ver -> Implementar nivel actual  expandirá solo el grupo actual (el grupo que contiene la fila actual). Esto no expandirá todos los subgrupos.  Ver -> Nivel de despliegue ... hará visible cualquier grupo de un cierto nivel, así como sus antepasados.

 También tenga en cuenta que si se produce un cambio en el texto plegado, se despliega automáticamente para mostrar el texto.

El bloque retráctil que contiene la línea activa se resalta en el margen. Puede cambiar el color específico utilizando el Configurador de coloreado de sintaxis al cambiar el estilo Global, Plegar activo.
