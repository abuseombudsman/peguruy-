Fichiers en lecture seule

Un document ouvert dans Notepad++ peut être marqué comme étant en lecture seule, ce qui signifie qu'il ne peut pas être modifié.

    Notepad++ reconnaît les deux états suivants:
  • Le fichier dans le système de fichiers est marqué comme en lecture seule. Vous ne pouvez activer l'édition du fichier en désactivant l'indicateur de lecture seule. Vous pouvez le faire en sélectionnant  Edition -> Enlever l'attribut Lecture-Seule du fichier .
  •  Le document dans Notepad++ est marqué comme en lecture seule. Vous pouvez régler manuellement cette option en sélectionnant  Edition -> Lecture seule pour le Document Actuel . Le document dans Notepad++ ne peut pas être modifié, mais si le fichier existe dans le système de fichiers, il ne sera pas marqué comme en lecture seule; c'est juste un état interne.
