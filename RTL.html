 De derecha a izquierda (RTL)

 Notepad ++ permite escritura tanto izquierda-derecha (LTR) como derecha-izquierda (RTL). La interfaz de usuario determina el diseño utilizando el archivo de traducción (si el archivo contiene el parámetro RTL = "yes" " para el elemento " Native-Language " , para una lectura a la derecha) dejó RTL, de lo contrario será la lectura izquierda-derecha (LTR).

 Cada vista también puede configurar de forma independiente el modo LTR o RTL, solo seleccione Pantalla -> Texto izquierdo a la derecha para mostrar texto LTR y  Pantalla -> Texto a la derecha Izquierda para mostrar el texto RTL . Tenga en cuenta que actualmente Scintilla tiene problemas con la visualización de texto mixto RTL / LTR.
