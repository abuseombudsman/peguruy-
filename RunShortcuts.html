 Accesos directos para el menú de ejecución

 Accesos directos predeterminados

 A continuación se muestra una lista de atajos de teclado predefinidos y accesos directos del mouse para Notepad ++ . Se pueden configurar libremente utilizando atajos de teclado ... , así como también cualquier comando que no esté relacionado con accesos directos predeterminados. Esta lista no incluye los accesos directos estándar proporcionados por los complementos de Notepad ++ .

Ejecutar menú
 atajo  acción
 F5  Iniciar diálogo de ejecución
 Alt-F1  Obtenga ayuda de PHP
 Alt-F2  Búsqueda de Google
 Alt-F3  buscar Wikipedia
 Alt-F5  Abra el archivo (nombre debajo del cursor)
 Alt-F6  Abra el archivo en una nueva instancia (nombre debajo del cursor)
 Ctrl-Alt-Shift-R  Abrir con Chrome
 Ctrl-Alt-Shift-X  Abrir con Firefox
 Ctrl-Alt-Shift-I Abrir con Internet Explorer
 Ctrl-Alt-Shift-F  Abrir con Safari
 Ctrl-Alt-Shift-O  Enviar a través de Outlook
