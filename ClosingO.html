Fermeture des fichiers

 Afin de fermer les fichiers que vous avez ouverts, vous disposez de plusieurs options.  N'oubliez pas que si vous avez des données non sauvegardées dans un fichier, Notepad++ vous demandera toujours si vous souhaitez enregistrer les modifications avant de fermer tous les fichiers.

 Principalement, vous pouvez fermer tout fichier en sélectionnant  Fichier -> Fermer  ou en cliquant sur .

 Pour fermer tous les fichiers, sélectionnez  Fichier -> Fermer tout  ou appuyez sur .

 Pour fermer plusieur fichiers simultanément, le menu Fermer... propose divers choix:

  • Fermr tout sauf le document actif
  • Ferler tous les onglets sur la gauche
  • Ferler tous les onglets sur la droite
'sur la droite', quand il y a plusieurs lignes d'onglets, s'applique aussi aux autres lignes.

Ces options sont aussi disponibles via le menu contextuel d'onglet, mais non regroupées en un sous-menu.

 Il y a aussi un paramètre qui permet de fermer un onglet en double-cliquant dessus, ou afficher un bouton de fermeture sur chaque onglet ().  Pour voir comment procéder, consultez la section sur la configuration de la barre d'onglets dans la boîte de dialogue Préférences.

 Enfin, vous pouvez fermer un onglet actif en cliquant sur la croix dans le coin en haut à droite de la barre de menu, ou encore, si vous activez cette option, par double clic sur l'onglet.
