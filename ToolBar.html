 Barra de herramientas

     La barra de herramientas proporciona acceso rápido a algunos comandos de menú (incluidos los complementos una vez guardados). Si la pantalla es más pequeña que el ancho de la barra de herramientas, algunos de sus botones están ocultos. Puede acceder a estos botones haciendo clic en el cheurón (la flecha doble) a la derecha de la barra de herramientas, aparecerá un menú que le mostrará los comandos ocultos. Puede personalizar la barra de herramientas para usar diferentes iconos. Notepad ++ admite tres modos:
  •  Iconos estándar: este es el valor predeterminado. Estos iconos son pequeños y también incluyen íconos de complemento.
  •  Iconos de Grangs: muestra iconos más grandes y se puede transformar en íconos personalizados. Los botones tienen diferentes imágenes cuando se seleccionan.
  •  Iconos pequeños: igual que los iconos grandes, pero el tamaño de la barra de iconos estándar.
    • Para elegir qué modo usar, consulte Preferencias .

    Para cargar imágenes personalizadas para su barra de herramientas, necesita un archivo de configuración especial e imágenes personalizadas. Para aprender a utilizar estos elementos, puede consultar el siguiente enlace para obtener información:
     http://sourceforge.net/apps/mediawiki/notepad-plus/index.php?title=Toolbar_Customisation
