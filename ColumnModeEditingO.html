Mode Colonne

Modification des zones rectangulaires

 Une autre façon d'éditer un fichier est de l'éditer en mode colonne. Cela signifie que le texte peut être sélectionné par colonnes, plutôt que par lignes (« édition verticale »).  Le support de l'édition en colonne se fait via Scintilla, mais avec quelques suppléments de Notepad++. Pour lancer le mode colonne pour la sélection, appuyez sur la touche Alt (par défaut) lors de la sélection de texte (en utilisant les touches de navigation ou la souris), en déplaçant le curseur vers le bas ou vers le haut selon les besoins. Cela fonctionne mieux si une police non-proportionnelle (exemple : Courrier New) est utilisée, car chaque caractère sera visuellement aligné sur celui de la même colonne au dessus et en dessous (voir Configurateur de coloration Syntaxique pour plus d'informations).

 Lorsque le texte est sélectionné dans ce mode appelé rectangulaire, la saisie au clavier changera chaque colonne de caractères sélectionné, y insérant le caractère saisi, ou bien insérera une nouvelle colonne. Utilisez la touche ESC pour sortir de ce mode de saisie spécial.  Pour reculer d'une colonne, utilisez les  options-> Annuler fonction Edit, et non la touche Retour arrière.

 Avant de taper "faire"  Après avoir tapé "faire"

La commande Edit -> Mode Colonne.... affiche un message pour rappeler l'existence de ce mode et les raccourcis clavier par défaut. De fait, ils n'apparaissent dans aucun autre menu.

 L'éditeur de colonnes

 La boîte de dialogue Editeur de colonne est conçue pour rendre le travail plus facile avec des colonnes. Il peut être ouvert en sélectionnant  Édition -> Édition en mode colonnes...

    Il existe deux types d'insertion pris en charge par l'éditeur en mode colonne, le texte normal ou des séquences de nombres.
  •  En sélectionnant insertion de texte, il insére sur chaque ligne le texte saisi, à la occolonne du curseur, de la ligne courante jusqu'à la fin du document.
  •  L' insertion de nombres va de même insérer sur chaque ligne une valeur numérique, selon les paramètres fournis:
    •  Le nombre initial est le premier à insérer.
    •   Le pas  : valeur dont on augmente la précédente avant la prochaine insertion.
    •  Si les Compléter par des zéros est cochée, les valeurs les plus petites seront complétées avec des zéros pour les rendre de même longueur que les plus grandes.
    •  Enfin, vous pouvez choisir dans quelle base les nombres seront générés, par défaut "décimal" est utilisé (base 10).
