 Informes de errores

 Si encuentra que Notepad ++ no funciona correctamente o simplemente se congela, es probable que tenga un error. Verifique primero si tiene la última versión, cada edición de Notepad ++ se mejora y los errores son corregidos (desafortunadamente, a veces agregamos algunos, entonces lo correcto es siempre tenga cuidado al actualizar). Además, si acaba de agregar o actualizar un complemento, podría ser la causa del error.

     Luego verifica si un plugin está causando problemas. Elimine todos los complementos o comience con la opción de línea de comando para deshabilitar todos los complementos y ver si el problema persiste. Si es el caso:
  •  Verifique los foros: quizás otros usuarios hayan encontrado este problema y hayan encontrado una solución.
  •  Compruebe el "rastreador de errores" ( http://sourceforge.net/tracker2/?group_id=95717&atid=612382 ): el error ya se puede detectar y en la lista. Averigüe si está en la lista; de lo contrario, puede agregarlo y ayudar a los desarrolladores a diagnosticarlo y solucionarlo (de la manera más clara posible: ¿cuál es el problema? causas y cómo reproducirlo Informes de error son, lamentablemente, actualmente muy inútil, una descripción sobre cómo reproducir el bloqueo es esencial. Si es así, intente también enviar zip o 7-zip el archivo NppDump.dmp si se produjo: este informe contiene el estado interno de Notepad ++ cuando ocurrió el accidente,
