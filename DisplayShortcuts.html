 Accesos directos para el menú Ver

 Accesos directos predeterminados

 A continuación se muestra la lista de teclas preestablecidas y accesos directos del mouse para Notepad ++ . Se pueden configurar completamente utilizando el correlacionador de acceso directo , así como cualquier comando que no esté relacionado con accesos directos predeterminados. Esta lista no incluye los accesos directos estándar proporcionados por los complementos de Notepad ++ .

Ver menú
 atajo  acción
 Ctrl- (Teclado Numérico - / +)  o el botón de la rueda del mouse Ctrl + (si corresponde) Zoom (+ o arriba) y Alejar (- o abajo)
 Teclado numérico Ctrl /  Restaurar el tamaño de zoom original
 F11  Modo de pantalla completa
 F12  Post-it (modo de palanca)
 Ctrl-Alt-F  Para recuperar el nivel actual
 Ctrl-Alt-Shift-F  Implementar el nivel actual
 Alt-0  Para recuperar todo
 Alt- (1 ~ 8)  Ocultar los niveles (1 ~ 8)
 Alt-Shift-0  Despliega todos los pliegues
 Alt-Shift- (1 ~ 8)  Dividir niveles (1 ~ 8)
