 Línea de comando

 Notepad ++ .exe [-alwaysOnTop] [-loadingTime] [-multiInst] [-noplugin] [-ro] [-nosession] [-notabbar] [-n ##] [c] [-x ##] [-y # #] [-l $$$] [-systemtray] {-r] [ archivos ] | [-openSession archivo ]

Texto original

Sugiere una traducción mejor
 Notepad ++ interpreta parámetros de línea de comandos para controlar su lanzamiento. Se reconocen las siguientes opciones (distinguen entre mayúsculas y minúsculas):
-alwaysOnTop
En presencia de esta opción, Notepad ++ se abre sobre otras ventanas.
 -c
 Comience editar en modo columna.
 --help
Mostrar esta lista, es decir, todos los conmutadores de línea de comando
 -l nombre abreviado del idioma
 Idioma uso para cada archivo abierto. $$$ es una cadena corta de identificación, aquí están los valores de autorizaciones:
 normal, PHP, C, CPP, CS, objc, d, java, RC, HTML, XML, Makefile, Pascual, mucho, ini, NFO, ASP, SQL, Visual Basic, JavaScript, CSS, Perl, Python, Lua, tex, COBOL, FORTRAN, golpe, actionscript, INE, TCL, Lisp, Scheme, asm, diff, accesorios, PostScript, Ruby, Java, VHDL, KIX, AutoIt, Gui4Cli, powershell, Caml, ADA, Verilog, Matlab, Haskell, la innovación, cmake, yaml, r, JSP, CoffeeScript
-L < local
Define el lenguaje utilizado para la HMI de la instalación que se abrirá. Lodal debe ser un LCID reconocido por Notepad ++, como en -Lru (Ruso) o -Les-ar (español argentino).
 -loadingTime
Muestra el tiempo de carga de todos los archivos si Notepad ++ comienza con esta opción. El resultado se muestra en un cuadro de mensaje, con una resolución de 0.01s.
 -multiInst
 Inicia más de una instancia de Notepad ++ . De forma predeterminada, si Notepad ++ ya está abierto, se realizará una nueva apertura en la misma instancia. La opción permite cambiar este comportamiento.
 -n < número de línea >
 Número de la línea actual, para todos los archivos abiertos al inicio.
 -noplugin
 No cargue complementos. Si sospecha que hay un complemento defectuoso o no desea cargarlos, agregue esta opción.
 -nosession
 No cargue la sesión anterior en el momento del arranque. No registre la sesión en la salida tampoco.
 -notabbar
 Oculta la barra de pestañas, incluso si las preferencias dicen lo contrario.
-openSession File
Abre archivo como un archivo de sesión, que carga todos los archivos a los que hace referencia.
 -r
¿Interpreta jokers? y * como lo haría DOS, que permite abrir varios archivos simultáneamente.
 -ro
 Cualquier archivo abierto al inicio estará en modo de solo lectura.
 -systemtray
 Inicie Notepad ++ minimis en la bandeja del sistema o en el área de notificación.
-x < posición >
 Especifique la posición horizontal (en píxeles) a la que se abrirá la ventana principal del Bloc de notas + + .
 -y < posición >
 Especifique la posición vertical (en píxeles) a la que se abrirá la ventana principal del Bloc de notas + + .
 archivos
 Todo lo demás se interpretará como nombres de archivo. ponga sus nombres de archivo entre comillas si contienen espacios. Los comodines no son compatibles.

 El orden de las opciones no importa. Por razones de compatibilidad, Notepad ++ primero tratará de identificar la línea de comando completo como nombre de archivo, incluso en ausencia de guilemets. Sin embargo, no se recomienda hacerlo, es siempre aconsejable colocar el nombre completo del archivo y cotizaciones.
