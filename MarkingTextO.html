Texte surligné

Surlignage de texte arbitraire

Lorsque du texte est sélectionné, l'utilisation de Rehcercher>Tout marquer->Appliquer le style xxx, où xxx vaut 1, 2, 3, 4 ou 5, surlignera toutes les ses occurrences du document actif.

Surlignage des résultats de recherche

Le marquage des réssultats de recherche se fait via l'onglet Marquer du dialogue Chercher/Remplacer. A cet effet, assurez-vous que l'option Marquer les lignes n'est pas cochée quand vous appuyez sur Rechecher tout. Les contrôles concernant ce sujet sont décrits dans Signets.

Lorsque le coloriage automatique est actif, la même recherche se fait implicitement sur la sélection (ou le mot sous l curseur si rien n'est sélectionné) à chaque modification de la séletion.

Dans les deux, c'est le Style de recherche qui est utilisé pour surligner les occurrences.

Configuration

Vous pouvez utiliser le Configurateur de colorisation syntaxique pour changer la couleur de n'importe lequel de ces styles, qu'il soit général ou lié à la recherche.

Autres commandes de surlignage

Vous pouvez vous déplacer au sein du texte surligné au moyen de Aller vers le haut ou Aller vers le bas>. Ces sous-menus permettent d'atteindre la zone précédente / suivante à être surlignée de l'un des 5 styles généraux ou du style de recherche.

Pour procéder au démarquage pour un certain style de 1 à 5, choisir Rechercher->Enlever tout->Retirer le style xxx. On ne peut enlever le style de recherche que par le bouton Effacer tous les styles du dialogue Rechercher / remplacer. On peut également suprimer tous les effets de style par Rechercher->Tout démarquer->Enlever tous les styles.
