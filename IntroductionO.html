Introduction

Nous dédions ce document aux innombrables heures perdues non seulemet par ceux qui ont cherché la réponse à des questions simples, mais aussi par les gens qui ont apporté des réponses à des questions qu'on ne devrait pas avoir à poser. La documentation est un art difficile, et une bonne documentation est pratiquement inaccessible =].

Toutes les images utilisées dans ce fichier d'aide sont prises avec les paramètres par défaut. Si vous modifiez un de ces paramètres, vous aurez un décalage avec  l'image, c'est surtout visible avec la barre d'outils (la position des images sera toujours le même).

Les boutons de la souris sont utilisés pour un droitier. Toutefois, pour une configuration en mode gaucher, vous devrez inverser l'utilisation des boutons. Les actions ordinaires dans Notepad++ sont essentiellement calquées sur l'utilisation standard de Windows.

De même, l'affichage standard est justifié de gauche à droite. Avec certaines localisations (comme l'Hébreu ou le Farsi), Notepad++ peut utiliser la justification inversée.

Ce balisage est fondé sur l'apparence par défaut de Microsoft Windows. Cela peut être différent sur votre propre système, mais les noms (texte) sont toujours les mêmes. Ceci est particulièrement important sur les captures d'écran. Elles sont seulement là à titre de référence, votre apparence de Notepad++ peut être très différente.
