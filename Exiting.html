Lanzamiento de Notepad ++

Para salir del Bloc de notas ++ , seleccione  Archivo -> Salir  o presione el botón Cerrar en la barra de título. Si tiene cambios no guardados, Notepad ++ le preguntará si desea guardar estos cambios (o cancelar el proceso de salida). Al salir, Notepad ++ guardará la sesión actual de manera predeterminada (ver Administración de sesiones ) para que se pueda volver a abrir en el próximo inicio. Para cambiar este comportamiento, mira la sección Preferencias .

    A veces, por una variedad de razones (un complemento defectuoso o un error del sistema operativo), Notepad ++ inesperadamente se interrumpe espontáneamente, se dice en modo bloqueado. En tal situación, él hace dos cosas:
  1. Intenta guardar los datos no registrados, si los hay. Los resultados son% TEMP% \ N ++ RECOV \. Notepad ++ muestra un cuadro de mensaje para informarle sobre el proceso.
  2. Al cierre, producirá un archivo binario de aproximadamente 300 Kb llamado NppDump.dmp. Enviar el archivo a los desarrolladores de Notepad ++ ayudará a comprender el accidente y corregir la anomalía tan pronto como sea posible, para que dichos eventos ocurran solo excepcionalmente. Si el culpable es un complemento, el autor del complemento puede ser notificado con una descripción general del accidente, él para que tome las medidas necesarias. Un plugin identificado como causa de accidentes se desactiva automáticamente; puede volver a ponerlo en acción si lo desea.
