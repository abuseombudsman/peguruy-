Ouverture et enregistrement de fichiers:

Ouverture

    Ouvrir... est très semblable à toute autre application Windows :
  •  Pour ouvrir un fichier, il suffit d'aller dans Fichier-> Ouvrir... , ou appuyez sur . Une boîte de dialogue apparaît et vous pouvez sélectionner un ou plusieurs fichiers qui doivent être ouverts.  Chaque fichier sera ouvert dans un onglet individuel.  Si le fichier est déjà ouvert, Notepad++ va vous montrer ce fichier dans son état actuel.  Dans la boîte de dialogue, un filtre vous permettra de ne montrer que les fichiers qui appartiennent à un langage donné (tels que C++ ou PHP) en fonction de l'extension.  Par défaut tous les fichiers sont affichés. Si vous tenez d'ouvrir plus de 200 fichiers, Notepad++ vous en avertira, au cas où ce serait une fausse maneuvre.
  •  Une autre façon d'ouvrir un fichier est de glisser le fichier dans Notepad++ en utilisant le curseur, sur la barre de titre de l'application.  Cela peut aussi être fait avec plusieurs fichiers ou dossiers sélectionnés ensemble.
  • De plus, Notepad++ conserve une liste des fichiers non ouverts, et récemment fermés (dans le menu Fichier) vous permettant de les rouvrir rapidement.  Il suffit de cliquer sur le nom du fichier pour l'ouvrir (voir la section sur la liste des fichiers récents pour plus de détails). Pour ouvrir tous les fichiers dans cette liste, il suffit de sélectionner Fichier-> Ouvrir tous les fichiers récents. Fichier-> Nettoyer liste des fichiers récents  permet de réinitialiser cette liste). Si la liste est contenue dans un sous-menu, ces deux commandes y figurent également.
  •  Voir aussi l'option ligne de commande pour voir comment ouvrir des fichiers de cette manière.

Enregistrement

    L'enregistrement d'un fichier peut se faire de plusieurs façons :
  •  Le mode par défaut est de sélectionner Fichier-> Enregistrer... ou d'appuyer sur .  S'il s'agit d'un fichier existant, il sera remplacé par le contenu actuel.  Notez que si le fichier n'a pas été modifié, vous ne pouvez pas enregistrer le fichier, l'option n'est pas accessible.
  •  Si vous souhaitez enregistrer tous les fichiers que vous avez ouverts à la fois, vous pouvez sélectionner  Fichier-> Enregistrer tout ou appuyer sur .
  •  Si le fichier n'a qu'un nom calculé (par exemple, si vous aviez créé un nouveau fichier), ou que vous avez sélectionné  Fichier-> Enregistrer sous , la boîte de dialogue d'enregistrement de fichier apparaît, vous demandant le nom du fichier.  Vous pouvez ensuite choisir l'emplacement et le nom du fichier.  L'extension .txt est ajoutée si le nom du fichier n'a aucune extension. Ajoutez un point pour éviter ce comportement.
  •  Une variante utile :  Fichier-> Enregistrer une copie.  La différence avec  Fichier-> Enregistrer sous est que votre travail se poursuit sur le nouveau fichier, et non pas sur le fichier origine.
  •  Le filtre de la boîte de dialogue peut ajouter automatiquement ou modifier l'extension pour vous. L'extension par défaut sera en tête de la liste du filtre.

Sauvegardes

Les sauvegardes, qui sont optionnelles, prennent deux formes:

  • A l'enregustrement d'un fichier, vous pouvez demander à Notepad++ de conserver une copie de sauvegarde. Le nom de cette copie comporte optionnellement un horodatage, et peut être placée dans un répertoire dédié plutôt qu'à côté du fichier enregistré.
  • Si Sauvegardes et enregistrements de session périodiques est cochée, le contenu de tous les fichiers ouverts dans leur état immédiat (non enregistré), ainsi que la session les décrivant, seront sauvegardés à intervalles réguliers, de lobgueur configurable. Il peut en résulter que votre fichier sur disque contienne son état enregistré le plus récent, alors qu'une copie instantanée plus récente existe ailleurs, et apparaîtra dans l'éditeur à sa réouverture. Ceci est particulièrement utile en cas de fermeture acidentelle du programme, mais peut surprendre si vous travaillez sur le même fichier avec d'autres outils que Notepad++. Notepad++ ne vous avertit pas à la fermeture que des fichiers ne sont pas enrgistrés; il faut se fier aux icônes d'état sur les oglets.
All options controlling what kind of backup is on and configuring backup are on the backupp tab in the Preferences dialog box

Other related operations

 Vous pouvez  Fichier->Recharger à partir du disque un fichier si vous souhaitez soit mettre à jour la vue actuelle du fichier (par exemple, si le fichier a été modifié entretemps) soit réinitialiser son contenu à ce qui est sur le disque (si vous avez modifié le fichier, mais pas encore enregistré). Si le fichier a été modifié sur le disque d'une autre manière en décalage avec le contenu en mémoire, Notepad++ va vous demander par défaut si vous souhaitez le recharger (voir Préférences pour plus de détails).

 Enfin, vous pouvez choisir de renommer un fichier en sélectionnant  Fichier-> Renommer . Une boîte de dialogue vous demandera le nouveau nom du fichier, semblable à la boîte de dialogue Enregistrer sous.  Toutefois, le fichier d'origine sera supprimé.

Le gestionnaire de projets

Cet outil, sous la forme d'une fenêtre ancrable, permet l'affichage sous forme arborescente d'ensembles de fichiers, et permet d'en ouvrir un par un dans la vue principale. On ouvre ces fenêtres par Affichage -> Projet -> Affichage de projet n où n vaut 1, 2 ou 3. Sous une racine appelée Workspace peuvent se trouver des projets, eux-mêmes composés de répertoires et fichiers.

Chaque élément a un menu contextuel. Celui de l'espace de travail est également affiché via WorkSpace, celui des autres éléments étant disponible via Edit.




L'espace de travail

Chaque espace de travail est lié à un fichier qui en décrit le contenu à la manière d'une session. Ainsi peut-on:



  • Remettre à zéro l'espace
  • Charger un fichier d'espace
  • Recharger le fichier d'espace (pour écraser des modifications indésirables)
  • Sauvegarder l'état de l'espace
  • Sauvegarder dans un autre fichier d'espace, et le rendre courant
  • Sauvegarder une copie dans un fichier d'espace (et continuer avec l'actuel)
  • Ajouter un nouveau projet

La touche Entrée, de même qu'un double clic ou clic simple sur le symbole d'escamotage s'il existe, change l'état d'escamotage de l'élément.

Au sein d'un espace de travail, tout élément peut être déplacé par glisser-déplacer, pour autant que cela ait un sens - déplacer un projet à l'intériur d'un répertoire n'a pas de sens. En outre, un élément peut être déplacé vers le haut ou le bas dans la catégorie qui le contient, et ce en sélectionnant l'élément puis en utilisant les touches Ctrl-Haut vers le haut ou Ctrl-Bas vers le bas.

Les projets

On peut renommer ou supprimer un élément via le menu contextuel, ou bien y ajouter des répertoires et des fichiers. Le changement de nom peut aussi se faire par clic sur l'étiquette, à la manière de l'explorateur de fichiers de Windows. La touche Suppr supprime aussi l'élément.

La commande Ajouter des fichiers depuis un Dossier... vous permet de choisir un répertoire. Ceci étant fait, tou ses fichiers sont ajoutés au projet, et tous ses répertoires sont ajoutés comme répertoires du projet.

Les répertoires

Les mêmes commandes sont disponibles. Un répertoire est un simple contenant, il n'est en rien lié à un répertoir de votre système de fichiers, même s'is ont été ajoutés via la commande Ajouter des fichiers depuis un Dossier... .

Les fichiers

Un fichier de l'espace de travail référence le fichier correspondant sur votre ordinateur. Les mêmes commandes sont disponibles, hormis Ajouter un répertoire; les fichirs n'ont pas de symbole d'escamotage. On peut en revanche changer le chemin complet du fichier via Modifier le chemin du fichier - cela permet par exemple de basculer d'une version à l'autre quand elles sont dans des répertoires différents sur votre ordinateur. La touche Entrée ou un double clic ouvre le fichier dans la vue active.

Si un nom de fichier ne correspond pas à un fichier réel, son icône est surchargée pour signaler ce fait, et on ne peut pas l'ouvrir.
