 Buscar sin cuadro de diálogo

 Investigación adicional

 Después de buscar un archivo, puede seleccionar Buscar-  > Buscar siguiente o  Buscar-> Buscar anterior  para rehacer esta búsqueda. Esto le permite realizar rápidamente la misma búsqueda varias veces sin necesidad de otra manipulación. El atajo de teclado es particularmente útil para usar.

 Del mismo modo, puede usar  Buscar-> Selección y Búsqueda siguiente  o Buscar-> Selección  previa y búsqueda  para navegar por los resultados que se muestran en la ventana de la r- Resultado de la búsqueda .

 Búsqueda volátil

    Una búsqueda volátil es buscar una palabra debajo del cursor sin tener que abrir el cuadro de diálogo de búsqueda. Esto tiene dos ventajas:
  •  Buscar -> Buscar volátil Siguiente  o  Buscar -> Buscar volátil  Búsquedas anteriores para la aparición siguiente / anterior del texto seleccionado, sin afectar las búsquedas actuales.
  •  Buscar -> Seleccionar y buscar en Siguiente o  Buscar -> Seleccionar y buscar Anterior  busca la ocurrencia anterior / siguiente aparición de la palabra debajo del cursor, y almacena esa búsqueda para que la investigación continúa con la misma palabra.
