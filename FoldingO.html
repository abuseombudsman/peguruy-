 Pliage / dépliage de texte

 Notepad++ prend en charge le pliage / dépliage pour vos documents texte. Cela signifie que certaines lignes de votre texte peuvent être cachées sous la forme de marques dans la marge. Cela fonctionne bien pour des langages structurés tels que C++ ou XML.  Des blocs de texte sont hiérarchiquement liés entre eux et une fois repliés (on dit ausi escamotés), seule la première ligne en demeure visible - s'ils ne sont pas à l'intérieur d'un bloc parent replié. Ce dispositif est efficace pour jeter un rapide coup d'œil à votre document et ne montrer que la partie intéressante du document. Déplier le texte l'affiche de nouveau en entier.

 Vous pouvez reconnaître le texte qui peut être plié en regardant la marge sur la gauche. Cette marge affiche des icônes et des lignes pour indiquer la structure du texte (l'apparence dans la marge peut se changer, voir Configurateur de coloration syntaxique et Préférences ). L'analyseur lexical courant détermine la structure (voir langages ).  Par défaut, les blocs sont représentés par des carrés et des lignes pour indiquer la structure des plis. Un nœud portant un signe moins ( ) indique que le texte peut être replié. Inversement, un nœud avec un signe plus () indique que le texte peut être déplié. Un texte escamoté est de plus indiqué par une ligne horizontale au dessus de la première ligne du texte replié, qui restera toujours visible.  Les lignes en marge délimitent les lignes de texte repliables.  Les lignes verticales (  ) qui indiquent les groupes de lignes escamotables ensemble se terminent par une petite ligne horizontale ( ) ou par une boîte.

    Il ya plusieurs façons de plier du texte.
  •  Le principal moyen est de cliquer avec le bouton gauche de la souris sur une case avec un signe moins. Cela replie tout le texte qui se trouve dans ce groupe.
  • Même chose avec appui sur la touche Ctrl pour replier le groupe, mais aussi tous les sous-groupes à l'intérieur. Cela signifie concrètement que lorsque vous ouvrirez ce groupe, tous les sous-groupes seront encore repliés.
  • En utilisant le menu:   Affichage -> Replier tous  repliera chaque groupe dans le document, minimisant le plus de texte possible.  Affichage -> Replier le niveau actuel  n'escamote que le groupe actuel (le groupe qui contient la ligne courante, n'importe où dans le groupe). Ceci ne repliera pas les sous-groupes.  Affichage -> Replier le niveau ... va escamoter le niveau sélectionné (de 1 à 8). Cela replie les groupes à un niveau particulier, mais pas tout les sous-groupes. Vous pouvez utiliser cette option pour afficher votre texte avec un « niveau de détail » donné.
    Pour déplier le texte plié, appliquez l'une des actions suivantes:
  • Cliquez sur la case avec le signe plus dans la marge des plis, affichant le contenu de ce groupe.
  • Cliquez sur la case avec le signe plus avec la touche Maj ou Ctrl enfoncée. Cela permettra de déplier le groupe et les sous-groupes plié.
  • En utilisant le menu:   Affichage -> Déplier Tous  dépliera chaque (sous-)groupe dans le document, ce qui rend visible le maximum de texte.  Affichage -> Déplier niveau actuel  ne développera que le groupe actuel (le groupe qui contient la ligne courante). Cela ne dépliera pas tous les sous-groupes.  Affichage -> Déplier niveau ... rendra visible tout groupe d'un certain niveau ainsi que leurs ancètres.

 Notez également que si un changement intervient dans le texte replié, il se déploie automatiquement pour afficher le texte.

Le bloc escamotable contenant la ligne active est mis en relief dans la marge. Vous pouvez changer la coueur spécifique utilisée à l'aide du Configurateur de Coloration Syntaxique en y modifiant le style Global style, Fold active.

Poir finir, signalons que les sessions mémorisent les points d'escamotage et leur état. Cela permet de s'arrêter de travailler sur un document avec des plis, et de le retrouver dans le même état en le rouvrant.
