 Signets

 Un signet est un marqueur spécial que l'on place seulement sur une ligne, dans la marge. Avec ceci, vous pouvez rechercher facilement les lignes et effectuer certaines opérations dessus.

 Ajout / Suppression de signets:

 Vous pouvez ajouter ou retirer un signet sur une ligne en cliquant sur la marge de signet en face de la ligne, ou en sélectionnant  Rechercher -> Signets -> Afficher / Masquer un signet .

 Pour supprimer toutes les signets

 Sélectionnez  Recherche-> Signets -> AEffacer tous les signets.

 Naviguer parmi les signets:

 Pour déplacer le curseur vers la ligne suivante contenant un signet, sélectionnez  Rechercher -> Signets -> Signet suivant , pour aller à la ligne précédente contenant un signet, sélectionnez  Rechercher ->Signets -> Signet précédent .

 Modification des lignes à signet:

     Vous pouvez appliquer une opération sur toutes les lignes marquées :
  •  Pour couper chaque ligne à signet, sélectionnez  Rechercher ->Signets -> Couper Lignes favoris . Chaque ligne sera coupée dans le fichier et ajoutée au presse-papier, afin de pouvoir être collés tous ensemble.
  •  Pour copier, sélectionnez  Rechercher->Signets -> -> Copier Lignes marquées  à la place.
  •  Pour remplacer chaque ligne signet avec le contenu du presse-papier, sélectionnez  Rechercher ->Signets -> Coller pour Remplacer lignes marquées.
  •  Pour supprimer toutes les lignes à signet / sans signet, sélectionnez Rechercher -> Signets -> Supprimer les lignes marquées ou ... non marquées, respectivement.
  • On peut enfininveser le marquage de toutes les lignes par Inverser les signets.

 L'option de suppression vous permet d'effectuer une recherche les premières lignes, de les marquer avec un signet en utilisant la recherche puis de retirez toutes celles fondées sur un certain critère.

Signets sur tous les résultats de recherche

L'onglet Marquer de la boîte de dialogue Cherchr / Remplacer permet de placer un signet sur chaque ligne portant un résultat d'une recherche, en utilisant le bouton Rechercher tout. Les contrôles sous le champ Rechercher fournissent des options pour ce faire:

  • Marquer les lignes ajoute un signet à chque ligne comportant une occurrence. Voir également Texte Marqué.
  • Effacer à chaque recherche effacera tous les signets et surlignements avant une nouvelle recherche. Si l'option est décochée, les signets issus de diverses recherches s'accumuleront.
  • Sélection seulement ne surligne ou ne place un signet que dans le texte sélectionné.

En outre, le bouton Tout Effacer supprime tous les signets et surlignages.
