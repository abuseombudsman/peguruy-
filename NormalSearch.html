    Puede buscar texto como en la mayoría de los programas usando el cuadro de diálogo Buscar / Reemplazar . Puede abrir este cuadro de diálogo de la siguiente manera:
  • Buscar -> Buscar (o presionando  ),
  • Buscar -> Reemplazar(o presionando  )
  • O Buscar-> Buscar archivos.

 En cada una de estas opciones, abra la pestaña correspondiente en el diálogo Buscar / Reemplazar . Este cuadro de diálogo siempre permanecerá sobre el texto, pero aún puede continuar editando el texto si está abierto, y puede configurarlo para que sea transparente (usando la opción Transparencia), cuando inactividad (por pérdida de concentración) o todo el tiempo. El control deslizante controla el nivel de transparencia. Tenga en cuenta que la transparencia solo está disponible desde Windows 2000.

 En el campo Buscar, escriba el texto que desea buscar. Este cuadro se rellena automáticamente con el texto seleccionado en progreso, o la palabra debajo del cursor, o el último patrón de búsqueda, cuando el cuadro de diálogo Buscar está abierto. La lista desplegable contiene elementos de investigaciones previas.

     Las siguientes opciones limitarán la búsqueda:
  •  Palabra completa solamente  : la búsqueda solo tiene éxito si el texto buscado constituye una palabra completa (por ejemplo: si la cadena encontrada está delimitada por blancos o signos de puntuación).
  •  Respete el caso : la cadena de búsqueda estará en mayúscula / minúscula. Si esta opción está desactivada, la búsqueda no distingue entre mayúsculas y minúsculas.
  •  El bucle  continúa en la parte superior del archivo cuando la búsqueda encuentra el final del archivo, independientemente de la ubicación inicial del cursor.
     El modo de búsqueda determina cómo se procesará la consulta:
  •  En modo normal , el texto se buscará literalmente.
  •  En el modo extendido , se interpretarán secuencias de caracteres especiales. En este caso, puede elegir entre:
    •  \\ - Barra invertida;
    •  \ t - carácter de tabulación;
    •  \ r - carácter CR;
    •  \ n - carácter LF;
    •  \ 0 - carácter NULL;
    •  \ x ## - Valor hexadecimal (entre 00 y FF);
    •  \ u #### - valor hexadecimal ampliado entre 0000 y FFFF, destinado a caracteres Unicode);
    •  \ d ### - valor decimal (entre 000 y 255);
    •  \o### - valeur octal (entre 000 et 377);
    •  \b######## - Valeur binaire (entre 00000000 et 11111111).
