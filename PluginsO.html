 Plugins

 Notepad++ est extensible grâce aux plugins.  Les plugins sont de grands ou petits ajouts à Notepad++ pour améliorer ses fonctionnalités. Notepad++ est livré avec quelques plugins (si vous utilisez l'installateur, vous pourrez choisir ceux à ajouter), mais vous pouvez toujours ajouter les vôtres ou en supprimer certains. Les plugins sont situés dans le répertoire principal  d'installation de Notepad++. Les plugins sont des fichiers DLL et il suffit de les retirer ou de les ajouter pour qu'ils soient chargés ou non.

 Lorsque Notepad++ démarre, il regarde dans son dossier de configuration les plugins et charge toutes les dll qu'il trouve. Plus tard, vous pouvez ajouter des plugins en utilisant Paramètres -> Importer - Importer plugins. Faites attention et assurez-vous que votre version de Notepad++ est à jour pour ajouter de nouveaux plugins et que le plugin est compatible avec Notepad++: certains très anciens plugins ne fonctionnent plus avec les nouvelles versions de Notepad++, il appartient au développeur du plugin de le mettre à jour ou non.

 Actuellement, Notepad++ ne se trouve que en version Unicode. Il y a eu cohabitation entre des versions ANSI et Unicode, et seule la première existait à l'origine. Le plugin doit correspondre à cette version, sinon il ne fonctionnera pas (Notepad++ vous avertit à ce sujet):. Les plugins peuvent se trouver n'importe où sur Internet, mais beaucoup figurent dans le projet Notepad++ Plugins sur sourceForge.net. Il y a même un plugin, appelé Plugin Manager, qui gère les détails d'installation et de version de vos plugins pour vous.

Si vous souhaitez installer les plugins standard, voici ceux qui sont fournis:

  • DSpellCheck : signale les éventuelles fautes de frappe et suggère des corrections
  • NppFTP : intègre à Notepad++ la gestion de fichiers distants via FTP
  • NppExport : exporte es fichiers RTF ou HTML à partir du document actif, avec sa coloration syntaxique
  • Plugin Manager : automatise l'installation et la mise à jour de la plupart des plugins. Plugin Central en fournit une liste complète.
  • Converter : conversion de chaînes de caractères entre ASCII, HEX, décimal et octal
  • Mime Tool : encodage et décodage pour base64, quoted printable, URL et SAML
A noter que n'importe lequel des plugins "officiels" peut être remplacé ou retiré, avec Plugin Manager ou manuellement.

 Si un plugin fonctionne  mal, Notepad++ va le décharger de façon à ce que l'erreur ne se propage pas, évitant ainsi toute perte de données.  Dans un tel cas, vous recevrez des informations sur le plugin en anomalie. Il faut faire connaître ces informations sur le forum Plugin de développement afin d'éliminer la cause du problème dès que possible.
