 Chargement de sessions

 Pour charger une session, il suffit d'aller à  Fichier -> Charger une session...  Lors de la sélection d'une session, tous les fichiers qu'elle contient seront ouverts et leur dernier état rétabli.

 Si vous avez défini une extension de session par défaut (par exemple, .Session) pour Notepad++, vous pouvez ouvrir une session comme n'importe quel autre fichier. Si le fichier porte l'extension d'une session, Notepad++ tentera de le charger comme tel (voir aussi la section concernant les préférences ). Il est très utile de choisir une telle extension, mais surtout n'utilisez pas .xml pour cela!

 Si vous avez défini une extension par défaut pour les fichiers de session, vous pourrez filtrer ces fichiers dans la boîte de dialogue « Chargement sessions ».
