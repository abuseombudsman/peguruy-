 Le dialogue Documents

 Le menu Fenêtre vous permet de sélectionner un fichier à partir de la liste des 10 premiers documents ouverts dans la vue active (voir aussi Vue Multiple ).  La sélection d'un document de la liste permet de l'activer :
 
 Vous pouvez ouvrir la boîte de dialogue Documents en sélectionnant  Documents -> Documents ....  Elle vous permet de gérer tous les documents ouverts actuellement.

 La boîte de dialogue Documents affiche une liste de tous les documents ouverts dans la vue active.  Vous pouvez sélectionner un ou plusieurs documents dans la liste.  Comme toute liste de contrôle Windows, sélectionner plusieurs en faisant glisser le pointeur de la souris ou en appuyant sur Ctrl ou Maj lors de la sélection ou utilisez les touches fléchées de votre clavier en maintenant éventuellement la touche Ctrl ou Shift enfoncée (si vous le faites, utilisez la touche la barre d'espace pour (dé)sélectionner).

 Vous pouvez activer un document en cliquant sur le bouton  Activer . Ceci fermera la boîte de dialogue.

 Sélectionner une ou plusieurs lignes puis appuyer sur le bouton  Enregistrer , va effectuer l'opération d'enregistrement, si possible, sur tous les documents sélectionnés; si un Enregistrer sous ... est nécessaire, rien ne se passera (voir Gestion des fichiers).

 Sélectionner une ou plusieurs lignes puis appuyer sur le bouton Fermer la(es) fenêtre(s), fermera le(s) document(s). Si un document contient des modifications, il vous sera demandé si vous voulez les sauvegarder.

Pour trier les documents en utilisant la boîte de dialogue, d'abord trier la liste en sélectionnant l'un des titres de colonne en haut (le choix de tri s'inverse à chaque click sur la colonne ).  Le tri sera effectif après appui sur le bouton  Trier les onglets  .

 Cliquez sur   OK  pour fermer la boîte de dialogue.
