 Configurador de coloreado de sintaxis

 El Configurador de resaltado se usa principalmente para controlar la apariencia del texto en Notepad ++ , pero también le permite modificar algunos de los otros aspectos que tienen un color configurable por el usuario.

     El diseñador de resaltado de sintaxis tiene dos listas. La lista izquierda  Idioma  contiene todos los idiomas conocidos, así como otros elementos con configuraciones de color:
  •  Estilos globales: controla los colores de la interfaz de usuario.
  •  Resultados: el texto de la ventana de resultados de búsqueda
  •  DOS) style: colorea el texto para mostrarlo como en una ventana de shell command.com / cmd.exe.

 La lista del  estilo correcto  contiene todos los estilos configurables para el idioma elegido.

 Para la mayoría de los idiomas, el papel de los elementos en esta lista debe ser obvio. Así que aquí están solo los elementos de configuración global y algunas excepciones conocidas.
 Validación:
  •  Guardar y cerrar : guarde toda la información de estilo y cierre el cuadro de diálogo.
  •  Cancelar : cancela todos los cambios y cierra el cuadro de diálogo. La misma acción que cerrar el cuadro de diálogo.
 Configuración de cuadro de diálogo:
 La transparencia  hará que el cuadro de diálogo sea translúcido. El control deslizante controla el nivel de transparencia. Tenga en cuenta que la transparencia solo está disponible desde Windows 2000.
 Temas:
 Un tema de estilo es una colección completa de propiedades de estilo (ver más abajo), para todos los idiomas y parámetros globales. En algún momento, un tema está activo, y puede cambiarlo en cualquier momento utilizando la lista desplegable en la parte superior del cuadro de diálogo. La lista se lee al inicio desde la subcarpeta del tema, pero puede agregarla un poco más tarde usando Parámetro -> Importar -> Importar archivo (s) de tema .
 Propiedades de estilo:
 Todos los estilos tienen propiedades asignadas a ellos. Cada estilo tiene propiedades que otros no tienen, como un color de fondo. Si una propiedad no está disponible, es simplemente gris.
 colores:
     Al hacer clic en el cuadro de color, se muestra un menú emergente con los colores predeterminados principales. Al hacer clic en cambiar el color. Haga clic en Más colores ... para especificar manualmente otro color. Cada cuadro de color tiene "clic derecho": un clic derecho impone o elimina el uso del color de estilo predeterminado. La eclosión aparece en la casilla para indicar el uso del color predeterminado.
  •  Color de primer plano : estos son los colores utilizados para el texto (color de los personajes).
  •  El color de fondo:   estos son los colores del fondo del texto
Fuentes:
  •  Nombre de fuente: la fuente que se usará al mostrar texto. Deje este campo en blanco para usar el valor predeterminado.
  •  Tamaño de fuente: tamaño de fuente (en puntos) usar cuando se muestra texto. Deje este campo en blanco para usar el valor predeterminado.
  •  Negrita : si quieres poner el texto en negrita.
  •  Cursiva  : si quiere poner en cursiva el texto.
  •  Soulign : si quieres enfatizar el texto.
