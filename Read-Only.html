Archivos de solo lectura

Un documento abierto en Notepad ++ se puede marcar como de solo lectura, lo que significa que no se puede editar.

    Notepad ++ reconoce los siguientes dos estados:
  • El archivo en el sistema de archivos está marcado como de solo lectura. No puede habilitar la edición de archivos desactivando el indicador de solo lectura. Puede hacerlo seleccionando  Editar -> Eliminar atributo de solo lectura del archivo .
  • El documento en Notepad ++ está marcado como de solo lectura. Puede configurar manualmente esta opción seleccionando  Editar -> Solo lectura para el documento actual . El documento en Notepad ++ no se puede editar, pero si el archivo existe en el sistema de archivos, no se marcará como de solo lectura; es solo un estado interno.
