Elementos GUISiguiente: Barra de estado

 Barra de pestañas:

    Cada vista de un documento tiene su propia pestaña (por defecto). La barra de pestañas le permite elegir entre diferentes documentos para organizarlos. Las opciones de configuración se pueden cambiar en Preferencias . La barra de pestañas muestra para cada archivo su nombre y un pequeño icono que indica el estado de este archivo. Un disco azul (  ) significa que el archivo no está modificado (o que es un documento nuevo). Si el disco está en rojo (  ), significa que el archivo se ha modificado y que estos cambios aún no se han guardado. Un disco gris ( ) significa que el archivo está marcado como de solo lectura y no se puede modificar (consulte Solo lectura). Si varias pestañas son visibles en la pantalla, pero le falta espacio:
  •  Si no activó la barra de pestañas de líneas múltiples en Preferencias , aparecerán dos flechas pequeñas a la derecha que le permitirán desplazarse por las pestañas para encontrar la pestaña. buena. Y si hay un espacio vacío, un doble clic en este espacio creará un nuevo documento.
  •  Si lo ha habilitado, varias líneas de pestañas serán visibles.

 La pestaña activa es la que opcionalmente tiene una línea de color (opción habilitada por defecto). Si dos vistas están habilitadas (ver Vista múltiple ), la pestaña con una línea coloreada es la pestaña que estará activa cuando se cambian las vistas. Puede controlar los colores de la barra de pestañas con la ayuda del resaltador de sintaxis .

 Puede abrir el menú emergente de la barra de pestañas haciendo clic con el botón derecho sobre una pestaña. Este menú contextual le permite acceder rápidamente a algunos comandos para administrar sus archivos, obtener y establecer las propiedades del archivo o mover la pestaña a otras vistas o ventanas. Tres.
