 Búsqueda incremental


 Para abrir el cuadro de diálogo Búsqueda incremental, seleccione  Buscar -> Búsqueda incremental  . Esta es una forma rápida de encontrar la primera aparición de texto y luego encontrar todos los resultados consecuentes.

  •  Puede colorear todas las ocurrencias con las opciones  Resaltar todo . El color se puede ajustar usando el Configurador de resaltado .
  •  Si no se encuentra texto, el cuadro de texto se muestra en color rosa, como navegar por los navegadores web.
  •  La opción  Match Match  permite buscar independientemente del caso.
  •  Los siguientes botones realizan los movimientos:  >  para la próxima aparición,  <  para la ocurrencia anterior.

 Finalmente, el botón  X  cierra el cuadro de diálogo de búsqueda incremental .
